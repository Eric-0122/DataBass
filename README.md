# DataBass

## File Structure  
automation_demo.sql - this file has a demonstration of a transaction that uses our procedure/function/triggers  
databass_db_backup.sql - this is the exported database backup file  
DataBass_schema.sql - this is the file that holds the creation of tables/automation  
.csv - these csv files are extra inserts fro the songs and accredations  
extrainserts.sql - a couple of extra inserts for songs and accredations  
intital_insert_data.sql - this file contains all of the original data inserts and is derived from a parser project that takes 
in the spotify million playlist json and converts it to insert into statements. Find the project code [here](https://github.com/SamMac55/dataLexer)  
queries.sql - these are the 5-10 required queries that demonstrate the database capabilites.
